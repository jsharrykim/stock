# 🎯 전문가 토론 — 신규 매수 전략 후보 발굴

## 토론 개요
- **주제:** 현재 A/B/C/D 4그룹 매수 전략과 중복/충돌되지 않는, 기술분석 데이터 기반 신규 매수 전략 후보 발굴
- **목적:** 백테스트 시뮬레이션과 전문가 토론을 통해 추가 가능한 신규 전략 도출
- **일시:** 2026-04-18
- **참고 자료:** CONTEXT.md, trackChanges.gs, updateInvestmentOpinion.gs

## 자료 분석 결과

### 사용 가능한 지표

| 카테고리 | 지표 | 현재 활용 |
|---|---|---|
| 추세 | MA200 | A/B/C/D |
| 모멘텀 | MACD hist (당일/전일/전전일) | C (골든크로스), D (미사용) |
| 과매도/과매수 | RSI, CCI | C (RSI>70), D (RSI<40 or CCI<-100) |
| BB 위치 | 종가 %B, 저가 %B | A, B, C |
| BB 폭 | BB Width, BB Width 60일 평균 | A (squeeze 비율) |
| 추세선 | LR Trendline값, LR기울기 | D |
| 시장 | VIX, IXIC이격도 | D, A/B/C 필터 |

### 미커버 공간

```
[MA200 상방]
  A → squeeze 대기 (BB 압축 + 저가%B ≤ 50)
  B → BB 하단 극단 (저가%B ≤ 5)
  C → 모멘텀 최강 구간 (종가%B > 80, RSI > 70)
  ??? → BB 중간대(30-70%) + MACD 반전 시작  [E후보]
  ??? → LR 추세선 저점 터치 (MA200 상방)    [F후보]

[MA200 하방]
  D → VIX 공포 + 과매도 + 추세선 바닥
  ??? → MACD 골든크로스로 추세 전환 포착    [G후보]
```

## 🟢 ROUND 2: 각 관점별 분석과 근거

### 후보 조건 초안

**E후보: MA200 상방 MACD 저점 반전 (C의 거울상)**
- MA200 위 + MACD 골든크로스 + 종가%B < 50 + RSI < 55
- 예상 EV 4~6%, 승률 70~78%

**F후보: MA200 상방 LR 추세선 저점 터치**
- MA200 위 + MA200×1.12 이내 + LR기울기>0 + 저가≤LR추세선×1.03
- 예상 EV 5~7%, 승률 75~82%

**G후보: MA200 하방 MACD 추세 전환**
- MA200 아래 + MACD 골든크로스 + LR기울기>0 + %B>30
- 예상 EV 2~4%, 승률 60~68% — 강한 추가 필터 또는 보류 필요

### 합의점

| 후보 | 위치 | 핵심 메커니즘 | 기존 충돌 | 수정 포인트 |
|------|------|--------------|-----------|-------------|
| E | MA200 상방 | MACD 저점 반전 | A/B 부분 중복 | A/B 미충족 시에만 발동 |
| F | MA200 상방 | LR 추세선 이중 지지 | 충돌 없음 | MA200×1.12 상한선 필수 |
| G | MA200 하방 | MACD 추세 전환 | D와 독립 | 추가 필터 또는 보류 |

## 🔴 ROUND 5 & ✅ 최종 도출안

### 핵심 결론 (3줄 요약)
1. **F그룹 1순위 도입** — D그룹 검증 지표를 MA200 상방에 재활용, 과적합 위험 낮음
2. **E그룹 백테스트 후 2단계 도입** — C의 거울상으로 이론적 근거 명확, 노이즈 필터 보강 필요
3. **G그룹 보류** — 메커니즘은 타당하나 50거래 샘플 확보 전 도입 불가

### 최종 진입 조건 상세

#### F그룹 (즉시 백테스트 및 도입 검토)
```javascript
// F그룹: MA200 위 + MA200 근접 + LR 추세선 저점 터치
const fCond1 = currentPrice > ma200;
const fCond2 = currentPrice < ma200 * 1.12;
const fCond3 = lrSlope > 0;
const fCond4 = candleLow !== null && lrTrendline !== null && candleLow <= lrTrendline * 1.03;
const fCond5 = (rsi !== null && rsi < 50) || (pctBLow !== null && pctBLow < 20);
const entryGroupF = !entryGroupA && !entryGroupB
                 && fCond1 && fCond2 && fCond3 && fCond4 && fCond5
                 && nasdaqAllowsAB;
```

#### E그룹 (백테스트 확인 후)
```javascript
// E그룹: MA200 위 + MACD 저점 반전 + BB 중하단
const eCond1 = currentPrice > ma200;
const eCond2 = macdHistD1 !== null && macdHistD1 <= 0 && macdHist !== null && macdHist > 0;
const eCond3 = pctB !== null && pctB < 50;
const eCond4 = rsi !== null && rsi < 55;
const eCond5 = bbWidth !== null && bbWidthAvg60 !== null && bbWidthAvg60 > 0
            && (bbWidth / bbWidthAvg60) < 1.5;
const entryGroupE = !entryGroupA && !entryGroupB && !entryGroupF
                 && eCond1 && eCond2 && eCond3 && eCond4 && eCond5
                 && nasdaqAllowsAB;
```
